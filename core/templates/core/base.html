Trabalho Final de Programação WEB - IFPI

Professor: Ritomar

Aluno: Elke Maclaine de Oliveira Sabino

Modulo: IV

Atividade Exigida

Desenvolver um programa para WEB com Python e Django para manipular uma ou mais tabelas de bancos de dados e utilizar template que faça uso de CSS e/ou Java Script. O programa deverá obrigatoriamente realizar operações de inserção, exclusão, alteração etc.

Atividades Realizadas

Foi realizado o que foi exigido, foi usado um template como front. O tema escolhido foi controle de receitas, onde no menu receitas na parte superior temos as opções de inserir uma nova receita como também editar e excluir.Foi usado CSS para estilizar alguns textos e controloar alguns botões como também alinhamentos. O campo de busca obtem resultados pelo nome do prato ou pelo tempo de preparo em minutos O site é responsivo
